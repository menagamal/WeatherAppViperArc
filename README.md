This a simple weather iOS app it was build using rxswift and rxcocoa 


# How to Run it !

  - Clone the project 
  - Navigate to the project 
  - run 'pod install'
  - And That's it .
  


# What is Viper !
VIPER is an application of clean architecture to iOS Apps

  - Interactor
  - Presenter
  - Entity
  - Router
  - View
